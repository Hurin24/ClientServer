Это тестовое задание. Приложение представляет из себя проект, который позволяет собрать исполняемые файлы в виде клиента и сервера.
Сервер позволяет принимать соединения на желаемом адресе. Получает символы и отправляет их обратно.
Клиент позволяет подключиться к желаемому серверу. Отправляет символы и ждёт их обратно.
